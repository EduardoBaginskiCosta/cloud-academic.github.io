Windows com .NET Framework

Devido aos requisitos das bibliotecas e estruturas do software, a Cloud Academic™ só será executado em sistemas Microsoft® Windows™ modernos. Os requisitos mínimos incluem:

• Windows XP SP3 ou superior Microsoft® .NET Framework 2.0 (x86 | x64).
• Bibliotecas de execução do Microsoft® Visual C++ 2008 Redistributable, com atualização de segurança ATL (x86).
