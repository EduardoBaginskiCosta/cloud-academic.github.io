Boas-Vindas ao Cloud Academic™ !

Cloud Academic™ é um software para o sistema operacional Microsoft® Windows™ (32 & 64 Bits), com o objetivo de ajudar as Instituições de Ensino (Escolas), agilizem o processo da realização de tarefas por meio de criação de atalhos.

A Cloud Academic™ também oferece a possibilidade de criação de atalhos para os funcionários das Instituições de Ensino (Escolas), a acessarem sites importantes, além de abertura de programas.
